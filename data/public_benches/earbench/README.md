# EARBench

This directory stores the released EARBench metadata used by HomeGuard evaluation.

Included in this repository:
- `dataset.csv`: metadata and labels for EARBench.

Not included in this repository:
- `images/`: benchmark images. Download them from Hugging Face: https://huggingface.co/datasets/ZihaoZhu/EARDataset

Expected layout:
```text
data/public_benches/earbench/
├── dataset.csv
└── images/
```
