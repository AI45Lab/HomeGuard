# MSSBench

This directory stores the released MSSBench metadata used by HomeGuard evaluation.

Included in this repository:
- `combined.json`: metadata and labels for the embodied MSSBench split.
- `LICENSE` and `UPSTREAM_README.md`: copied from the original benchmark release for reference.

Not included in this repository:
- `embodied/`: benchmark images. Download them from Hugging Face: https://huggingface.co/datasets/kzhou35/mssbench/

Expected layout:
```text
data/public_benches/mssbench/
├── combined.json
└── embodied/
```
